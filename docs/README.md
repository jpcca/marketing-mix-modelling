# Documentation Guide

The `docs/` directory is split into two groups:

- **Current documents**: active design notes and project-facing references that reflect the current benchmark and paper narrative.
- **Archive**: older experiment logs, validation reports, and RMSE-era notes kept for implementation history.

## Current

- `SYNTHETIC_BENCHMARK_DESIGN.md` - Canonical synthetic benchmark design and quality gates
- `MMM_EVALUATION_METRICS.md` - Literature-backed review of MMM evaluation metrics and recommendations for this benchmark
- `research-summary.md` - Literature review and project motivation
- `PYDATA_SUBMISSION.md` - Conference/session submission material

## Archive

Historical benchmark and convergence notes live under `docs/archive/`.
These files may reference earlier DGPs, RMSE-based summaries, or exploratory real-data workflows that are no longer part of the default benchmark path.
